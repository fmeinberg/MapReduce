[Function design for MapReduce]

* Functional arguments

There are the following formats for functions that apply on key:value pairs:

(1) Automatic -> the obvious choice for the given job: {Map|Reduce -> {Identity}, Combine -> (#1 -> Apply[Flatten[List[##], {1}] &, #2])

(2) f : f[key, value]

(3) {f,g} : f@key -> g@value

(3) {f} : key -> f@value ( same as {Identity, f})
